# Pact Community Organization Foundation

Welcome to the Foundation repository of the Pact Community Organization (PCO). This repository serves as the authoritative source for the organization's governance, mission, vision, and operational processes.

## Mission

To make it easy and safe for businesses to start building with smart contracts.

## Vision

A vibrant blockchain ecosystem where builders confidently deploy trusted, reusable smart contracts maintained by a transparent, community-driven, nonprofit organization.

## Contents

- **Governance**: Learn how decisions are made and roles are defined in `governance/`.
- **Policies**: Standards and processes for operations in `policies/`.
- **Community**: How to get involved and contribute in `community/`.
- **Documentation**: Organizational policies and public-facing resources in `docs/`.
- **Onboarding**: Complete guide to joining and advancing in the organization in `docs/ONBOARDING.md`.

## Repository Structure

- `docs/`: Public documentation and GitHub Pages site.
- `governance/`: Governance documents, maintainers, and decision-making processes.
- `policies/`: Security, support, funding, and operational policies.
- `community/`: Contribution guidelines and code of conduct.
- `foundation.wiki/`: Wiki content for the organization.
- `wiki/`: Additional wiki resources.
- `.github/`: GitHub workflows and templates.

## Onboarding and Maintenance

See the docs and wiki for onboarding and maintenance instructions. New community members should start with [ONBOARDING.md](docs/ONBOARDING.md) for a complete guide to participation and role progression.

## Get Involved

We welcome contributions from the community! Please see our [CONTRIBUTING.md](community/CONTRIBUTING.md) for details on how to get started.
